# 3.4 Exploratory Data Analysis (EDA)

## Objective

Analyze the prepared dataset to understand customer behavior, identify patterns, detect anomalies, and generate insights before training Machine Learning models.

---

## Analysis Tasks

The following analyses will be performed:

- Dataset overview
- Data types inspection
- Missing value analysis
- Duplicate record analysis
- Statistical summary
- Distribution analysis
- Correlation analysis
- Outlier detection

---

## Customer Analytics

Analyze customer behavior using:

- Customer distribution
- Purchase frequency
- Spending patterns
- Geographic distribution

---

## Sales Analytics

Analyze:

- Order trends
- Revenue distribution
- Payment methods
- Product categories

---

## Visualization

The following charts will be generated:

- Histogram
- Box Plot
- Bar Chart
- Line Chart
- Correlation Heatmap
- Pie Chart

---

## Expected Outcome

The EDA process will identify:

- Important features
- Data quality issues
- Customer behavior patterns
- Business insights
- Inputs for feature engineering and model development
