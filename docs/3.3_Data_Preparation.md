# 3.3 Data Preparation

## Objective

Prepare the raw Olist dataset for Machine Learning by cleaning, transforming, and integrating data into a single analytical dataset.

---

## Data Preparation Steps

### 1. Data Loading

- Load all required CSV files.
- Verify data types and file integrity.

---

### 2. Data Integration

Merge required tables using common identifiers such as:

- customer_id
- order_id
- product_id
- seller_id

---

### 3. Data Cleaning

Perform the following operations:

- Handle missing values
- Remove duplicate records
- Correct inconsistent data types
- Remove unnecessary columns

---

### 4. Feature Engineering

Generate new features including:

- Recency
- Frequency
- Monetary (RFM)
- Total Orders
- Total Spending
- Average Order Value
- Delivery Time
- Review Score

---

### 5. Target Variable Creation

Create target variables for Machine Learning:

- Churn Label
- Customer Segment
- Customer Lifetime Value (CLV)
- Purchase Propensity

---

### 6. Data Transformation

- Encode categorical features
- Scale numerical features (where required)
- Convert date columns to datetime format

---

### 7. Final Dataset

The prepared dataset will be stored in the `data/processed/` directory and used for model training and evaluation.
