# 4.3 Data Integration

## Objective

Integrate multiple Olist tables into a unified analytical dataset for Machine Learning.

## Tables Integrated

- Customers
- Orders
- Order Items
- Order Payments
- Order Reviews
- Products
- Sellers

## Output

A consolidated customer-level dataset named `customer_orders_master.csv` is generated in the `data/processed/` directory.

## Note

Geolocation and Product Category Translation tables are excluded from the initial integration as they are not required for the first version of the Machine Learning pipeline.
