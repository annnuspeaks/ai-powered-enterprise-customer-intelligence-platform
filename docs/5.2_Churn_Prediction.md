# 5.2 Churn Prediction

## Objective

Train a supervised Machine Learning model to predict whether a customer is likely to churn.

## Algorithm

- Random Forest Classifier

## Input Features

- Recency
- Frequency
- Monetary
- Total Orders
- Total Spent
- Average Order Value
- Average Review Score
- Average Delivery Days

## Target Variable

- Churn

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Output

- Trained model: `models/churn_prediction_model.pkl`
