# 2.2.4 UI Wireframe

## Screen Overview

The platform consists of the following primary screens:

- Dashboard
- Dataset Upload
- Prediction Center
- Reports

---

## Dashboard

Displays business insights and key performance indicators.

Components:

- KPI Cards
- Revenue Overview
- Customer Segmentation Chart
- Churn Analysis
- CLV Distribution
- Purchase Propensity Summary

---

## Dataset Upload

Allows users to upload customer datasets.

Components:

- Upload CSV Button
- Dataset Validation Status
- Upload Progress
- Dataset Summary

---

## Prediction Center

Allows users to generate Machine Learning predictions.

Components:

- Churn Prediction
- Customer Segmentation
- CLV Prediction
- Purchase Propensity Prediction
- Prediction Results

---

## Reports

Displays generated reports and export options.

Components:

- Business Summary
- Prediction Reports
- Export CSV
- Export PDF (Future)

---

## Navigation

- Dashboard
- Upload Dataset
- Prediction Center
- Reports

---

## UI Design Principles

- Clean and professional interface
- Responsive layout
- Interactive charts
- Consistent color scheme
- Simple navigation
- User-friendly experience
