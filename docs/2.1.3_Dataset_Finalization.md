# 2.1.3 Dataset Finalization

## Selected Dataset

**Olist Brazilian E-Commerce Public Dataset**

---

## Dataset Source

- Platform: Kaggle
- Domain: E-Commerce
- Type: Public Dataset

---

## Dataset Description

The Olist Brazilian E-Commerce Dataset contains real-world e-commerce transactions from multiple marketplaces in Brazil. It includes customer information, orders, products, sellers, payments, reviews, and geographical data.

The dataset is relational and consists of multiple linked tables, making it suitable for end-to-end customer analytics and predictive modeling.

---

## Reason for Selection

The dataset was selected because it:

- Contains real-world business data.
- Supports customer-centric analytics.
- Enables multiple Machine Learning tasks.
- Provides relational data for feature engineering.
- Is suitable for Business Intelligence dashboards.
- Is widely used for academic and portfolio projects.

---

## Machine Learning Modules Supported

The selected dataset will be used to develop the following modules:

- Customer Churn Prediction
- Customer Segmentation
- Customer Lifetime Value (CLV) Prediction
- Purchase Propensity Prediction

---
