# 5.1 Customer Segmentation

## Objective

Group customers into similar behavioral segments using K-Means clustering.

## Algorithm

- K-Means Clustering

## Features Used

- Recency
- Frequency
- Monetary

## Output

- Cluster label assigned to each customer.
- Clustered dataset saved as `ml_dataset_clustered.csv`.

## Customer Segments

- Active Customers
- High Value Customers
- Loyal Customers
- At-Risk Customers
